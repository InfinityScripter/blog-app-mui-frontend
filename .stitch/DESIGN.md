# Design System: aifirst.us.com — «Editorial Ink» (target)

Premium redesign target. Generated by taste-design skill 2026-07-02.
Direction: Russian tech-editorial — warm paper, dense ink, one vermilion
signal color. Constructivist restraint instead of template gloss.
Taste dials: Creativity 8 · Variance 6 · Motion 5 · Density 5.

## 1. Visual Theme & Atmosphere

A reading-first AI journal that feels like a well-set print publication that
happens to be interactive: warm paper canvas, near-black warm ink, hairline
rules, and a single vermilion signal color used with discipline. Hierarchy
comes from typographic weight and spacing, not from colored boxes. Dark mode
is a warm charcoal darkroom — same ink logic, inverted.

Layouts are confidently asymmetric: left-anchored hero, featured-plus-list
feed, zig-zag content bands. Motion is physical and quiet — spring-driven
entrances, staggered list reveals, tactile button presses. Nothing shimmers,
nothing glows.

## 2. Color Palette & Roles

### Foundation (stone family — one temperature everywhere, warm)

- **Paper** `#FAFAF9` — light canvas (page background)
- **Pure Surface** `#FFFFFF` — cards, sheets, popovers (light)
- **Warm Ink** `#1C1917` — primary text (light scheme); NEVER `#000000`
- **Graphite** `#57534E` — secondary text, metadata
- **Ash** `#A8A29E` — disabled, placeholders
- **Hairline** `rgba(28,25,23,0.10)` — borders, rules, dividers (light)
- **Darkroom** `#141210` — dark canvas
- **Charcoal Surface** `#1C1917` — dark cards; **Soot Neutral** `#262220` — dark neutral fill
- **Bone** `#F5F5F4` — primary text (dark scheme)

### Accent — exactly one

- **Signal Vermilion** `#C2451E` (light scheme) / `#E06032` (dark scheme) —
  links, primary CTA, active nav, focus rings, selection. Ramp:
  lighter `#FBE3D9` · light `#E8794F` · main `#C2451E` · dark `#9A3412` · darker `#5C1F0F`.
- Old Minimals green `#00A76F` and violet `#8E33FF` are RETIRED.

### Functional states (semantic only, never decorative)

- Success `#15803D` · Warning `#B45309` · Error `#BE123C` · Info `#0E7490`

## 3. Typography Rules

- **Display: Unbounded** (Cyrillic-native, Google Fonts) — h1/h2 only.
  Weight 600, tracking −0.02em, line-height 1.02–1.1. Controlled scale:
  h1 clamp(2.5rem → 4rem), h2 clamp(1.75rem → 2.5rem). Hierarchy through
  weight + vermilion accents, not size screaming.
- **Structure & body: Manrope** (Cyrillic, geometric-humanist) — h3–h6
  weight 700/600 tracking −0.01em; body 400/500, line-height 1.6,
  max 65ch measure on article text.
- **Mono: JetBrains Mono** — dates, reading time, view counts, model
  versions, code, timeline years. Numbers in dense contexts always mono.
- Replaces: Public Sans (no Cyrillic — silent system fallback today) and
  Onest. Inter BANNED. Generic serifs BANNED.

## 4. Component Stylings

- **Buttons:** flat vermilion fill (primary) / hairline outline (secondary),
  radius 10px, weight 700 Manrope, active state translateY(1px) tactile push,
  hover darkens fill ~8%. No glows, no gradients.
- **Cards:** radius 20px, 1px hairline border, shadow only `0 1px 2px
rgba(28,25,23,0.06)` at rest; hover lifts translateY(−2px) with tinted
  diffuse shadow `0 12px 24px −8px rgba(28,25,23,0.12)`. In dense lists
  replace cards with border-top rules + whitespace.
- **Post feed item:** cover with radius 16px, mono date + tag chip row,
  Manrope 700 title that shifts to vermilion on hover (no underline),
  graphite excerpt, hairline separator between items.
- **Chips/tags:** hairline outline pill, mono 12px uppercase, vermilion
  text when active; never filled color blocks.
- **Inputs:** label above, radius 10px, hairline border, 2px vermilion focus
  ring; error text below in `#BE123C`. No floating labels.
- **Loaders:** skeleton shimmer blocks matching layout; circular spinners
  only inside buttons.
- **Nav (header):** paper-translucent blur bar with bottom hairline; links
  Manrope 600; active = vermilion text + 2px underline offset 6px; cmd-K
  search as outlined pill with mono `⌘K` hint.
- **Footer:** ink-on-paper columns, hairline top rule, mono small print.

## 5. Layout Principles

- Max content width 1200px (`Container lg`); article measure 720px.
- Hero: left-aligned asymmetric split — headline block left (7/12), meta
  panel right (5/12); NEVER centered; no scroll cues.
- Feed: featured post large (spans 2 cols) + vertical list; no 3-equal-cards
  rows anywhere — use 2-col zig-zag or 1+list composition.
- 8px spacing grid; section rhythm `clamp(4rem, 9vw, 7rem)`.
- Single-column collapse below 900px (md); touch targets ≥44px;
  no horizontal scroll on mobile ever.
- CSS Grid via MUI Grid; no calc() percentage hacks; no overlapping
  absolute stacks — every element owns its zone.

## 6. Motion & Interaction

- Framer Motion springs: UI `stiffness 260 / damping 30`; section reveals
  `stiffness 100 / damping 20`. No linear easing.
- Staggered cascade on feed/timeline mount (60–80ms steps, once).
- Perpetual micro-motion, subtle only: pulsing live-dot on «свежее»
  badge, slow float on hero ornament. No text shimmer.
- Animate `transform`/`opacity` only. Respect `prefers-reduced-motion`
  (global CSS kill-switch already present).

## 7. Anti-Patterns (BANNED)

- Emojis in UI · Inter · generic serifs · pure `#000000`
- Neon/outer glows · oversaturated accents · second accent color
- Gradient text on large headers (current hero shimmer — REMOVE)
- Centered hero · 3-equal-card feature rows · overlapping collage stacks
- «Scroll to explore», bouncing chevrons, scroll arrows
- AI copy clichés («Elevate», «Seamless», «Next-Gen», «Unleash»)
- Fabricated metrics/round numbers; placeholder names
- Custom cursors · `h-screen` (use `100dvh`) · broken Unsplash links
- Cool/warm grey mixing — stone family only

## 8. Keep Intact (engineering constraints)

CSS-vars theme pipeline (`extendTheme`, class color-schemes, no-flash
script), settings drawer (primary presets must still work — default preset
becomes vermilion), RHF\* form components, MUI v7 Grid, kebab-case files,
≤200 lines/file, hooks in `hooks/`, sections isolation, `prefers-reduced-motion`.
